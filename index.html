Symmetry

Detects the part's vertical mirror planes (magenta lines). Lay flat first for best results — lines follow every rotation.

2 Mark pin & hole

Pin and hole are shown as vertical cylinders. Each next link is placed so its hole lands on the previous link's pin.

3 Build chain

Archimedean spiral r = a + bθ. Start radius a is the innermost radius (offset so it doesn't start at the centre). Loop spacing is the radial gap between successive turns — keep it bigger than the part width so loops don't collide.

Joint anchors (live)

Nudge the pin / hole to retune link pitch & body offset until overlaps (red) clear — the chain re-lays as you type.

Size variation (per part)

JS f(n_it, n_it_nor) → scale. n_it = part index, n_it_nor = that index normalized to 0–1. Edits apply live; the chain re-spaces so joints still mate. 1 = original size.

4 Export

All links merge into one mesh as separate, non-touching solids — a print-in-place articulated chain.
